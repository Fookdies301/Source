# customRules

## 阅读 App 书源

来自公众号 `开源阅读软件`，`BookSource.json` 为公众号分享的群友在线书源，1101个书源为本地书源（由书友 Nightring 收集分享）。

虽然书源十分多，高达一千多个，但是我认为贵在精而不在多，而且分享的源中还有许多失效的书源，便决定自己搜集整理一下。做个简单的分类。
